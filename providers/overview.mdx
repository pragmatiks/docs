Providers extend Pragmatiks to manage different data tools.

## Available Providers

### GCP

Google Cloud Platform resources.

| Resource | Description |
|----------|-------------|
| `gcp/storage` | Cloud Storage buckets |
| `gcp/bigquery-dataset` | BigQuery datasets |

### Pragma (Built-in)

Platform resources.

| Resource | Description |
|----------|-------------|
| `pragma/secret` | Secure credential storage |
