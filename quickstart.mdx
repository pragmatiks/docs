## Prerequisites

- Python 3.11+
- A Pragmatiks account

## Step 1: Install the CLI

```bash
pip install pragma-cli
```

## Step 2: Authenticate

```bash
pragma auth login
```

This opens your browser for authentication.

## Step 3: Create a Resource

Create a file called `bucket.yaml`:

```yaml
provider: gcp
resource: storage
name: my-bucket
config:
  location: EU
```

## Step 4: Apply the Resource

```bash
pragma resources apply bucket.yaml
```

## Step 5: Check Status

```bash
pragma resources list
```
