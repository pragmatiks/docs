## Requirements

- Python 3.13+

## Install

```bash
pip install pragmatiks-cli
```

Or with pipx:

```bash
pipx install pragmatiks-cli
```

## Verify

```bash
pragma --version
```

## Shell Completion

### Bash

```bash
eval "$(pragma --show-completion bash)"
```

### Zsh

```bash
eval "$(pragma --show-completion zsh)"
```

### Fish

```fish
pragma --show-completion fish | source
```
