The Pragmatiks API provides programmatic access to all platform features.

## Base URL

```
https://api.pragmatiks.io
```

## Authentication

All API endpoints require authentication using Bearer tokens.

```bash
curl -H "Authorization: Bearer <token>" \
  https://api.pragmatiks.io/v1/resources
```

Get a token by running `pragma auth login` or through the web dashboard.

## OpenAPI Specification

The full API specification is available in OpenAPI 3.0 format. Use it to generate client libraries or explore endpoints.

## Rate Limits

API requests are rate limited. Contact support if you need higher limits.
