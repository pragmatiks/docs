Pragmatiks is an orchestration layer for data tools. Define your infrastructure as YAML, and let providers manage the lifecycle.

## What Pragmatiks Does

- **Declarative configuration**: Define resources in YAML files
- **Lifecycle management**: Create, update, and delete resources automatically
- **Dependency tracking**: Resources wait for their dependencies before processing
- **Provider ecosystem**: Extensible architecture for any data tool

## Quick Example

```yaml
provider: snowflake
resource: warehouse
name: analytics-wh
config:
  size: MEDIUM
  auto_suspend: 300
```

```bash
pragma resources apply warehouse.yaml
```
