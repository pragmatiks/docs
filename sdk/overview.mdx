The Pragmatiks SDK provides a Python client for interacting with the API programmatically.

## Installation

```bash
pip install pragma-sdk
```

## Basic Usage

```python
from pragma_sdk import PragmaClient

client = PragmaClient()

# List resources
resources = client.list_resources(provider="gcp")
for resource in resources:
    print(f"{resource['name']}: {resource['lifecycle_state']}")

# Apply a resource
result = client.apply_resource({
    "provider": "gcp",
    "resource": "storage",
    "name": "my-bucket",
    "config": {"location": "EU"}
})
```

## Authentication

The SDK uses the same credentials as the CLI. Run `pragma auth login` first, or set the `PRAGMA_AUTH_TOKEN` environment variable.
